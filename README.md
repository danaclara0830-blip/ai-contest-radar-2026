# AI 콘텐츠 공모전 레이더 2026

국내/국외 AI 영상, 이미지, 광고, 아트, 솔루션 공모전을 모아 보는 정적 HTML 웹앱입니다.

## 기능

- 마감 지난 공고 자동 숨김
- 국내/국외, 접수중, 마감임박, 분야별 필터
- 검색, 정렬, CSV/JSON 내보내기
- 카드별 공모 주제, 제출 콘텐츠, 추천도, 일정 진행률 표시
- 즐겨찾기와 개인 메모를 브라우저 localStorage에 저장

## 배포

Vercel 정적 사이트로 배포합니다. GitHub 저장소의 `main` 브랜치가 업데이트되면 Vercel Git Integration으로 자동 배포됩니다.
